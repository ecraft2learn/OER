23 STARTING A PRINT When the build plate has been leveled and the filament has been loaded, you can start your (first) print. We’ve placed some print files on the SD card to start with. Simply select one of the files and press the button to start. After a print file has been chosen, the Ultimaker 2+ will prepare itself by homing the print head and build plate and heating up the build plate and nozzle. Please note that this can take up to 5 minutes. While printing, the display will show the progress of the print and the remaining time for completion. When the print has finished, wait for the build plate to cool down and grab the print from the build plate. Do not touch the print head/nozzle while it is heating, printing or cooling down. Temperatures can reach up to 260°C. For good platform adhesion it is recommended to apply a thin layer of glue to the glass plate. For this, you can use the glue stick delivered with the Ultimaker 2+. Refer to “USING GLUE” on page 25 for detailed instructions. STARTING A PRINT
