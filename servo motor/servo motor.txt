<h1 style="text-align: justify;">Servo motor</h1>
<h3 style="text-align: justify;">A type of DC motor that allows for precise control of angular position. It consists of a&nbsp;motor&nbsp;coupled to a sensor for position feedback. It also requires a&nbsp;servo&nbsp;drive to&nbsp;complete the system. The drive uses the feedback sensor to precisely control the&nbsp;rotary position of the&nbsp;motor. It can be used for tasks that require precise motion (e.g., a&nbsp;robot arm).</h3>
