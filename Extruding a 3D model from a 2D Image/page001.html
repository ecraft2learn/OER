EXTRUDING A 3D MODEL FROM A 2D IMAGE Importing an image into Cura Open Cura and in the top menu, select File, then Open File(s). Browse to the image you wish to import, select it then click the Open button. The Convert Image window is then shown. Set the required Height, this is the maximum extruded height of the print in millimetres. Set the required Base, this is the base height of the print in millimetres, set this to 0.0 if you do not want to print a base. Set the required Width or Depth of the print in millimetres. When one of these dimensions is entered, the other is calculated in proportion to the size of the original image. Select Darker is higher or Lighter is higher. This selection will make either the darker areas of the image print higher or the lighter areas of the image to print higher. For the settings shown in the example above, the black areas of the image will print 10mm high and the white areas of the image will print at 1mm high. Cura can import .JPG .PNG .BMP and .GIF fle types. The minimum and maximum heights of the image can be set by the user, then Cura uses the gray- scale of the image to set the height of diferent areas of the image (within the minimum and maximum dimensions).
