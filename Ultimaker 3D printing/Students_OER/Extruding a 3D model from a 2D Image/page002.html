2D to 3D Examples The following examples show some of the ways this feature of Cura can be used along with the import settings to convert the images. 1. Nozzle Size 2. Bookmark In this example as darker is higher, the text will at 2mm thick, the bookmark shape is 50% grey so it will print at 1mm thick (so the bookmark is fexible) and the white areas don’t print at all, creating holes as the base is set to 0.0mm. grey would print at a height of 5.5mm. Set the required Width or Depth of the print in millimetres. When one of these dimensions is entered, the other is calculated in proportion to the size of the original image. Drag the Smoothing slider to select how much smoothing to apply between the colour transitions. Click OK, this will import the image into Cura and convert it into a 3D model with the selected settings. The 3D model can then be manipulated in Cura and prepared for 3D printing in the same way as standard 3D models. Image Cura import settings 3D print
